# Projektmodul Web — Course Repository

This repository is the central submission repo for the course **Projektmodul Web**.

Each team must upload and maintain their project files in **their assigned team folder**. Please do **not** place files in the repository root.

---

## Submission Rules

- **Only commit to your team’s folder**.
- Add a short `README.md` inside your team folder describing:
  - Project title
  - Team members
  - How to run / build
  - Any dependencies or setup steps

---

## What Not to Upload

- Do not commit secrets or credentials (API keys, passwords, tokens).
